# British Equities Portfolio Management

## Overview
This project presents the construction and analysis of a British Equities Portfolio focused on undervalued FTSE 100 companies. The portfolio was developed using portfolio management theories, value investing principles, and top-down investment strategies to achieve long-term capital growth while managing portfolio risk effectively.

## Objectives
- Build an optimized UK equities portfolio
- Analyze portfolio risk and return characteristics
- Apply portfolio theory and diversification strategies
- Evaluate market sectors and investment opportunities
- Compare portfolio performance against the FTSE 100 benchmark

## Investment Strategy
The portfolio follows:
- Active portfolio management
- Top-down stock selection approach
- Long-term value investing philosophy
- Diversification across multiple sectors

## Key Sectors Included
- Property Development
- Consumer Goods
- Utilities
- Industrial & IT
- Transportation
- Healthcare

## Financial Models & Analysis
The project includes:
- Efficient Frontier Analysis
- Sharpe Ratio Evaluation
- Value at Risk (VaR)
- Tracking Error Analysis
- Covariance & Correlation Matrix
- Portfolio Optimization
- Bloomberg PORT Analytics

## Key Findings
- The portfolio achieved an expected return of 2.12%
- Diversification significantly reduced unsystematic risk
- The portfolio outperformed the FTSE 100 benchmark
- Defensive utility stocks reduced overall portfolio volatility
- Value investing opportunities were identified within undervalued UK equities
